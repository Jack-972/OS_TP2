# TP2 OS - Jacques ARNAULD & Adam BOUNOUR
Objectif : Réaliser un protocole en s’inspirant du fonctionnement très simple de NetBEUI, et de l’utiliser dans une application qui va permettre à chacun de communiquer avec les autres étudiants, ceci en utilisant les librairies aux standards POSIX qui ont été définies pour les systèmes de la famille Unix, dont GNU/Linux fait partie.
